**Email Splitter Pro** is a high-performance, multi-threaded Windows desktop application designed to process, filter, and split massive email datasets (1M+ records) with extreme speed and precision.

> **Note:** This repository serves as the official **Public Release & Issue Tracker** for Email Splitter Pro. The source code is maintained in a separate private repository.

---

## ✨ Features

- **⚡ Blazing Fast Engine:** Powered by a modern Rust backend, processing hundreds of thousands of records per second using highly optimized file buffering and multi-threading architecture.
- **🧹 Advanced Data Filtration:** Automatically cleans your data by removing empty lines, invalid email formats, and duplicate records in real-time.
- **📊 Domain Intelligence:** Instantly categorizes and displays statistics for major domains (Gmail, Outlook, Yahoo, Corporate) as your files are processed.
- **📈 Professional Reporting:** Automatically generates beautiful, interactive HTML dashboard reports for every split operation, detailing metrics, engine settings, and filtration stats.
- **🔄 Smart OTA Updates:** Built-in Over-The-Air updater automatically notifies you of new releases and seamlessly installs them directly from this repository.
- **💻 Premium UI:** A stunning, responsive interface built with React, Vite, and Tailwind CSS, featuring a sleek dark mode and fluid animations.

## 📥 Download & Installation

1. Go to the [Releases](../../releases/latest) page.
2. Download the latest `Email-Splitter-Pro-vX.X.X-x64-setup.exe` file.
3. Run the installer and follow the quick setup wizard.
4. Launch Email Splitter Pro from your Desktop or Start Menu!

> **⚠️ Windows Defender / SmartScreen Warning:** 
> Because this is a new, independently developed application, Windows and your web browser might show a warning like *"Microsoft Defender SmartScreen prevented an unrecognized app from starting"*. 
> **To bypass this:**
> - **In Edge/Chrome:** Click the `...` next to the download -> **Keep** -> **Show more** -> **Keep anyway**.
> - **When running the app:** Click **More info** -> **Run anyway**.

### System Requirements
- **OS:** Windows 10 or Windows 11 (64-bit)
- **RAM:** Minimum 4GB (8GB recommended for processing files larger than 100MB)

## 🛠️ How to Use

1. **Select Files:** Simply select your large `.txt` file containing emails and choose your output directory.
2. **Real-Time Scanning:** The app instantly runs a dry-run scan to show you exactly how many valid emails, duplicates, and domains are in your file *before* you split it.
3. **Configure Settings:** Choose your chunk size, toggle strict validation, or enable comma appending.
4. **Split!:** Hit the Start Processing button. The app will generate chunked files and a detailed HTML report in your chosen output directory.
